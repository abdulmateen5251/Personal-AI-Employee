# Odoo Tools

## odoo_list_partners

List partners from Odoo using JSON-RPC `execute_kw` with `search_read` on `res.partner`.

## odoo_health_check

Validate login/session and return current Odoo user context.

## odoo_accounting_readiness

Check journals/accounts availability and return readiness suggestions.

## odoo_list_journals

List accounting journals with type and code.

## odoo_list_accounts

List active chart-of-accounts entries.

## odoo_ensure_partner

Find existing partner by name/email or create one if missing.
